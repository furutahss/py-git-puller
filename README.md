# py-git-puller

指定したディレクトリの直下にある複数のGitリポジトリを一括で `git fetch` および `git pull` するための自動化ツールです。

## 概要

複数のプロジェクトを並行開発していると、個別に `git pull` を行う作業が大きな手間になります。

このツールは、指定した親ディレクトリ内のすべてのフォルダを走査し、Gitリポジトリを自動で検知して最新の状態に更新します。

## 機能

- **一括更新**: `git fetch --all` と `git pull` を各プロジェクトに対して順次実行します。
- **耐障害性**: 特定のリポジトリでエラー（競合、未コミットの変更など）が発生しても、それを無視して次のプロジェクトの処理を継続します。
- **自動検知**: ディレクトリ内に `.git` フォルダが存在するかどうかで、Gitリポジトリかどうかを自動判別します。
- **シンプル表示**: 更新が必要だったか、すでに最新だったかを簡潔に報告します。

## 使い方

以下のコマンドを実行します。

```bash
python main.py [リポジトリが並んでいる親ディレクトリのパス]
```
